Como o computador entende um programa?

linguagens de programação são linguagens usadas para a comunicação com o computador. Estas linguagens são constituídas de comandos, que quando utilizados corretamente, executam ações. As instruções colocadas em um programa são compiladas ou melhor dizendo traduzidas para o computador.

Algoritimo

É a especificação de passos em ordem lógica que visa resolver o problema proposto. Sem um problema não é possível desenvolver um algoritimo.

Um algoritimo complexo sempre terá uma falha que tera de ser retificada e da mesma forma que funciona um programa.

Ao resolvermos falhas em nosso programa, estamos criando novas versões do mesmo.

Para todas as estruturas temos Início e Fim (Begin e End), conceito este que usamos no pseudocódigo para que o aprendiz saiba que tudo que for aberto deve ser fechado.

8 comandos básicos para a criação de um algoritmo:

  • Entrada de dados: Leia
  • Saída de dados: Escreva
  • Processamento de dados
  • Condições: se/então se/então/senão se encadeado
  • Escolha
  • Estrutura de repetição: pré condicional: enquanto-faça pós condicional: faça-enquanto contador: Para
